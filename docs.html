Getting Started

Motion CSS is a minimal, modern CSS animation library with 100+ ready-to-use animations. No JavaScript required ‚Äî just add the CSS classes to your HTML elements and watch them animate.

Installation

Add the CSS file link in your <head>:

npm i motion-css-library

Basic Usage

Simply add the animation class to your element:

<div class="fade-in">Hello, world!</div>

Animations trigger on page load or when the element appears.
